## Install

Você irá precisar do [Node.js](https://nodejs.org/en/download/) e [Yarn](https://classic.yarnpkg.com/en/docs/install/) para rodar corretamente.

### Development Mode

```bash
yarn install
yarn dev
```

### Production Mode

```bash
yarn install
yarn build
yarn start
```
